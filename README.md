# Full Stack AI Content Platform with Next JS, Tailwind, React Quill, ImageKit, Shadcn UI  🔥🔥
A modern, high-performance SaaS platform built to generate, manage, and optimize AI-driven content. This project leverages the speed of Next.js 15, the real-time capabilities of Convex, and a sleek interface powered by Shadcn UI.

📂 Project Structure

├── app/              # Next.js App Router (Pages & API)
├── components/       # Reusable UI components (Header, Sidebar, Editor)
├── convex/           # Backend schema, functions, and triggers
├── hooks/            # Custom React hooks (Convex mutations/queries)
├── lib/              # Utility functions and third-party configs (ImageKit)
├── public/           # Static assets (Logo, Banners)
├── middleware.js     # Route protection and auth logic
└── package.json      # Dependencies and scripts
